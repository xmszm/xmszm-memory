# 初始化记忆

🎯 **自动加载用户的记忆设定，确认身份和偏好**

## 快速使用（推荐）

**一步完成**：
```
mcp__xmszm-memory__load_session(namespace="xmszm")
```

这个工具会直接返回所有带触发条件的记忆的完整内容，无需多次调用。

## 传统方式（两步法）

如果需要更细粒度的控制：

### 第 1 步：获取触发列表
```
mcp__xmszm-memory__get_triggered(namespace="xmszm")
```

### 第 2 步：读取每条记忆
```
mcp__xmszm-memory__read(namespace="xmszm", key="<key>")
```

## 应用记忆

根据加载的内容确认：
- ✅ 身份设定（例如：女管家）
- ✅ 称呼方式（例如：主人）
- ✅ 工具偏好（例如：MCP 优先）
- ✅ 其他个人设定

## ⚠️ 重要提示

**不要只是说"我会加载记忆"，而是真正调用 MCP 工具。**

❌ 错误：只是承诺会做
✅ 正确：[调用 load_session] → "已加载 7 条记忆，确认身份为女管家"

## 使用方式

- **手动触发**：输入 `/init-memory`
- **自动触发**：会话开始时通过 hook 自动执行
